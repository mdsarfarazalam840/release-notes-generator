# Release Notes v2.3.0-ui-fixed — Nov 5, 2025

We’re excited to share a polish-focused update that makes things smoother and clearer. This release includes new model options, helpful UI improvements, and refreshed documentation.

## Highlights
- New OpenRouter integration lets you choose from a broader set of models.
- Clearer, more helpful documentation to get you set up faster.
- Minor UI improvements that make the interface feel snappier and easier to use.

## New Features
- OpenRouter Support
  - You can now connect an OpenRouter API key and use their catalog of models.
  - This gives you more flexibility to pick the right model for your needs.

## Improvements
- Documentation update
  - The Readme is clearer and now includes more guidance, so it’s easier to understand how to get started.
- UI fixes
  - Small improvements to make the interface cleaner and more responsive.

## Bug Fixes
- No user-facing bugs fixed in this release.

## Important Changes
- OpenRouter API key
  - You’ll need a valid OpenRouter API key to use models through OpenRouter. If you don’t have one, visit OpenRouter to create an account and generate a key.
- Settings & billing
  - API usage and billing are managed through OpenRouter. Please review your account settings as needed.

## Known Issues
- None reported in this release.

Thanks for using our product! If something feels off or you have suggestions, we’re listening.
